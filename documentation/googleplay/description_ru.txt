Andlytics собирает статистику из Панели разработчика Google Play. Оно позволит вам отслеживать активные установки, все установки, оценки и комментарии для всех ваших приложений, опубликованных в Google Play.

Andlytics также может собирать статистику AdMob, включая выручку, запросы и мнения. Поддерживает синхронизацию в фоне, показывает уведомления при наличии изменений.

Google не предоставляет публичного интерфейса-API для получения данных о приложениях. Поэтому Andlytics может перестать работать, если Google поменяет что-либо в панели разработчика. В этом случае вы можете подождать, пока мы внедрим изменения.

Andlytics - приложение с открытым кодом, вы можете помочь проекту на GitHub.

Это приложение никак не связано с компанией Google.

Аккаунты "Google Apps for Business" и множественные соединенные аккаунты разработчиков  пока не поддерживаются. Если у вас один из таких аккаунтов, свяжитесь с нами через GitHub и помогите нам добавить поддержку этих функций.

Аккаунты AdMob могут исчезать на некоторых устройствах, перемещение приложения на SD-карту может решить проблему.

Для аккаунтов AdMob, присоединенных к аккаунтам Google, вы можете найти пароль API на странице информации об аккаунте http://www.admob.com/my_account/account_info.
