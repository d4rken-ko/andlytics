Andlytics récupère les statistiques de la console de développement Google Play. Cela vous permet de suivre le nombre d'installations actives, d'installation total, les notes, et les commentaires pour toutes vos applications Android publiées sur Google Play.

Andlytics peut également récupérer les statistiques AdMob, incluant le revenue, le nombre de requêtes et le nombre d' impressions. Andlytics permet de synchroniser les statistiques en tache de fond et d'afficher une notification quand un changement survient.

Google ne fournit pas de clé publique de son API pour collecter les statistiques. En conséquence, si Google modifie son site, cela peut provoquer un dis-fonctionnement d'Andlytics. Dans ce cas, merci d'être patient pendant que nous corrigeons le problème.

Andlytics est open source, n'hésitez pas à contribuer via GitHub.

Andlytics n'est en aucune façon affilié à Google.

"Google Apps for Business" et les comptes développeur Multi-connectés ne sont pas gérés pour le moment. Si vous possédez un de ces types de compte, contactez nous via GitHub et aidez nous à l'implémenter.

Sur certains dispositifs, le compte AdMob disparaît. Déplacer Andlytics depuis la sdcard vers le stockage interne peut régler le problème.

Pour les comptes AdMob qui sont liés au compte Google, vous pouvez trouver la clé d'API sur la page d'information du compte http://www.admob.com/my_account/account_info.
